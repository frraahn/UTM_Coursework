# 🚀 Business Intelligence & Data Engineering Portfolio

Welcome to my repository documenting my practical work, data workflows, and assignments completed using Alteryx in my Business Intelligence course.

---

## 📂 Repository Architecture & Project Breakdown

### 🔹 Course Project: Student Performance & Lifestyle Analytics
* **General Summary:** A group project focused on building an end-to-end data pipeline using Alteryx Designer. We combined multiple datasets containing student information (like study hours, sleep, screen time, and mental health) into one clean master file. From there, we built an analytical model to see exactly how daily habits affect a student's GPA and created interactive dashboards to highlight "at-risk" students.
* **Core Deliverables:**
  * Alteryx Data Cleaning and ETL Workflows (`.yxmd` files)
  * Integrated Master Dataset and Structured Data Models
  * Final Analytical Dashboards and Reports (Tableau / Power BI)

### 🔹 Class Exercises: Lab Modules & Alteryx Practice
* **General Summary:** A collection of weekly hands-on laboratory exercises designed to build strong skills in Alteryx. These tasks focused on moving away from manual spreadsheet work and moving toward automated data preparation. The exercises covered everything from basic data cleaning to advanced statistical analysis and forecasting.

## 💡 Academic Reflection

Working on these Alteryx workflows has completely changed how I think about data. Before this class, I mostly analyzed data manually using basic spreadsheets. Moving to an automated tool like Alteryx taught me how crucial it is to properly clean and prepare data before trying to find insights.

I learned how to solve real data problems, such as fixing messy datasets, joining tables without losing information, and grouping data to find hidden patterns. This experience helped me look at data from a bigger perspective. I now feel much more confident in my ability to take raw, messy data and turn it into clear, useful insights that can help organizations make better decisions.
