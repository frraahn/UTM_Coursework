# 🤖 Artificial Intelligence

Welcome to my repository for the **Artificial Intelligence** course at Universiti Teknologi Malaysia (UTM)! This portfolio documents our comprehensive project lifecycle alongside core assignments detailing the technical implementation of an intelligent agent framework.

---

## 📂 Project & Assignment Breakdown

### 🔹 Project: Smart Waste Management System (Smart City)
* **Summary:** Our team designed an AI and IoT-driven solution to tackle urban waste inefficiencies caused by fixed collection schedules. Following a design-thinking methodology from proposal to final architectural modeling, we conceptualized an intelligent system that monitors real-time bin levels, predicts waste accumulation trends, and dynamically automates optimized truck routing to prevent overflow and save fuel.
* **Files:**
  * Design Thinking Proposal (`/SECJ3553-02 ARTIFICIAL INTELLIGENCE PROPOSAL.pdf`)
  * Comprehensive Project Final Report (`/SECJ3553-02 ARTIFICIAL INTELLIGENCE (AI)_FinalReport.pdf`)

### 🔹 Assignments: Intelligent Agent Frameworks & Formulations
* **Summary:** A structured series of assignments focused on defining and mapping the logic components powering our smart city agent. The work involved building concrete Knowledge Representations (KR) using First-Order Logic (FOL), mapping out localized decision workflows using directed State Space Searches, and establishing operational boundaries using the formal PEAS model.
* **Files:**
  * **Assignment 1: Knowledge Representation & FOL Rules** (`/SECJ3553-02 ARTIFICIAL INTELLIGENCE (AI).pdf`)
  * **Assignment 2: State Space Search Formulations** (`/A2_AI_Group6.pdf`)
  * **Assignment 3: PEAS Model & Goal-Based Agent Behaviors** (`/A3_SECJ3553-02_ARTIFICIAL_INTELLIGENCE_(AI).pdf`)

## 💡 Course Reflection & Key Takeaways

This course completely reshaped how I think about solving complex, dynamic problems. Building our **Smart Waste Management System** proved that AI is far more than just writing code; it requires creating structured, reliable logical frameworks that allow machines to safely reason and act in unpredictable, real-world environments.

I gained valuable hands-on experience structuring **Knowledge Representations** and breaking down system milestones into executable steps using **State Space Searches**. Modeling our agent under the **PEAS framework** helped me clearly map out how raw sensor readings convert directly into precise backend actuator commands. Overall, this course gave me the essential foundation, critical thinking skills, and technical confidence needed to design smart, automated city systems that solve real human challenges.
